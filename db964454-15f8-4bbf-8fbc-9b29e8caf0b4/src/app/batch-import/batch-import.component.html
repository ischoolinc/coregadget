使用說明
  1. 從excel中複製成績
    • 請先確認excel中學生的人數及順序皆與系統相符
    • 複製成績時，僅需圈選成績的部分，不含學生姓名及表頭
    • 如有學生沒有成績資料，請輸入"-"，表示無資料，否則系統將認定錯誤
  2. 於下方文字方塊中貼上
  3. 使用"解析"按鈕，進行解析
  4. 預覽資料確認成績正確
  5. 使用"匯入"按鈕，進行匯入
    • 如果輸入資料有錯誤，將無法匯入
